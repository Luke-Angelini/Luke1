Logical Operators!

These types of operators don't necessarily deal with equivalent/non-equivalent values, but they rather work on operands to produce a singular boolean result

AND : returns TRUE if the operands around it are TRUE OR : returns TRUE if at least one operand is TRUE NOT : returns TRUE if the following boolean is FALSE Turn the following arithmetic phrases into either True or False statements as indicated USING LOGICAL OPERATORS
