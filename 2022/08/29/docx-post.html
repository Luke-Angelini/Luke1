Docx Files:

  • Docx Files can be created in Google Docs by downloading it as a Microsoft word document or by using Microsoft Word directly

  • Then the file should be saved using the `YYYY-MM-DD-*.dock` into the `/_word` folder.

  • Finally changes should be Synced with Github
